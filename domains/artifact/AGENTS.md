# artifact

Назначение: Blob refs, previews, screenshots, traces, verify reports и diff attachments.

Когда идти сюда:
- Когда задача касается хранения тяжёлых артефактов и их preview.

Соседи: session, git, browser, harness, search

Карта:
```text
artifact
├─ AGENTS.md
└─ docs/
   └─ context.mmd
```

Текущее состояние:
- Материализована карта домена.
- Код появится по owning slices, когда домен станет активным фронтом реализации.
