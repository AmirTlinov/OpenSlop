# Reference images

Здесь лежат канонические изображения для visual review.

Правило сравнения:
- не пиксель-в-пиксель;
- по смыслу, hierarchy, плотности, native поведению и общей форме окна.

Текущий минимальный набор:
- `workbench-shell-empty.svg` — пустой shell с читаемой anatomy;
- `workbench-shell-live.svg` — активный shell с inspector и narrative center;
- `workbench-shell-inspector-hidden.svg` — shell без inspector как layout state.

Честная пометка:
- это semantic references, а не screenshot baseline;
- они задают анатомию окна и visual grammar для ранних slice review;
- будущий screenshot automation живёт позже, в visual-conformance tooling.
