# OpenSlop

OpenSlop — нативный macOS workbench для Codex, Claude Code и будущих агентных движков под одним быстрым GUI.

Сейчас в репозитории заложен первый load-bearing слой:
- конституция проекта;
- доменные карты;
- flagship roadmap по вертикальным слайсам;
- минимальный buildable seed для macOS app;
- минимальный buildable seed для core daemon;
- репозиторный shape-check.

Стартовое чтение:
1. `AGENTS.md`
2. `PHILOSOPHY.md`
3. `ARCHITECTURE.md`
4. `DESIGN.md`
5. `ROADMAP.md`

Быстрые команды:
- `make doctor`
- `make smoke`
